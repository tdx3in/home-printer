# Changelog

All notable changes to the Home Printer Documentation site will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-28

### Added

- Initial release of Home Printer Documentation
- Complete documentation set:
  - Home Printer Basics
  - Common Features
  - Installation & Configuration Guide
  - Settings Reference
  - Maintenance Guide
  - Troubleshooting Guide
- Docusaurus v3 static site generator
- GitHub Pages deployment automation
- GitHub Actions workflow for CI/CD
- Responsive design supporting all devices
- Dark mode support
- Full-text search capability
- SEO optimization
- Comprehensive README and guides

### Project Structure

- 7 documentation pages covering printer setup and maintenance
- Custom styling and branding
- Modular sidebar navigation
- Static asset management (logos, images)
- Multiple deployment options
