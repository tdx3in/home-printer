# Common Home Printer Features

Home printers vary by model, but many include the same core features.

## Feature Summary

| Feature | What it does | Typical use |
| --- | --- | --- |
| Duplex printing | Prints on both sides of paper | Save paper for drafts and long documents |
| Borderless printing | Prints to the edge of the page | Photos and flyers |
| Scanner | Captures paper documents as files | Forms, receipts, IDs |
| Copier | Duplicates a physical page | Quick copies without a computer |
| Mobile printing | Prints from a phone or tablet | Photos, tickets, email attachments |
| Maintenance tools | Cleans print heads and aligns cartridges | Fix streaks or faded output |

## Choosing Features

- Use **duplex printing** for everyday documents.
- Use **high quality** only when the final appearance matters.
- Use **draft mode** for internal notes.
- Use **mobile printing** when the file is already on a phone.
- Use **scan to PDF** for contracts, forms, and receipts.

## Feature Availability

Feature names differ by manufacturer. If a feature is not visible in the print
dialog, check the printer driver, manufacturer app, or embedded web interface.
