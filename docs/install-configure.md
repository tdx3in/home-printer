# Install and Configure a Home Printer

Use this task to connect a home printer to a computer or mobile device and
confirm that it can print successfully.

## Before You Start

- [ ] Place the printer on a stable surface.
- [ ] Connect the power cable.
- [ ] Install ink or toner.
- [ ] Load plain paper.
- [ ] Turn on the printer.

## Install the Printer

1. Open the printer setup menu on the printer screen.
2. Choose **Wi-Fi**, **Wireless Setup**, or **Network Setup**.
3. Select your home Wi-Fi network.
4. Enter the Wi-Fi password.
5. Wait for the printer to show a connected status.
6. On your computer, open the printer settings.
7. Choose **Add printer**.
8. Select the printer from the list.
9. Install the recommended driver if prompted.

## Configure Defaults

| Setting | Recommended value |
| --- | --- |
| Paper size | Letter or A4, depending on region |
| Print quality | Normal |
| Color mode | Color |
| Duplex | On, if supported |
| Default tray | Main tray |

## Print a Test Page

1. Open the operating system printer settings.
2. Select the new printer.
3. Choose **Print test page**.
4. Confirm that the page prints clearly.

## Result

The printer is installed, connected, and ready for normal use.

:::note
Expected state:
- Printer appears in the device printer list.
- Printer status is online or ready.
- Test page prints without errors.
:::
