# Home Printer Settings Reference

Use this reference to choose common printer settings.

## Print Settings

| Setting | Common values | Recommended default | Notes |
| --- | --- | --- | --- |
| Paper size | Letter, A4, Legal, 4x6 | Letter or A4 | Match the loaded paper. |
| Orientation | Portrait, Landscape | Portrait | Use landscape for wide tables or images. |
| Color mode | Color, Grayscale, Black and white | Color | Use grayscale to reduce color ink use. |
| Quality | Draft, Normal, Best | Normal | Best mode uses more ink or toner. |
| Duplex | Off, Long edge, Short edge | Long edge | Requires duplex support. |
| Copies | 1 or more | 1 | Check before printing large jobs. |

## Network Settings

| Setting | Purpose |
| --- | --- |
| IP address | Identifies the printer on the network. |
| Hostname | Friendly network name for the printer. |
| Wi-Fi status | Shows whether the printer is connected. |
| Signal strength | Helps diagnose wireless reliability. |

## Status Values

| Status | Meaning | Action |
| --- | --- | --- |
| Ready | Printer can accept jobs | No action required |
| Offline | Device cannot reach printer | Check power and network |
| Paused | Queue is stopped | Resume the queue |
| Paper jam | Paper is stuck | Clear jam path |
| Low ink or toner | Supplies are nearly empty | Replace soon |
