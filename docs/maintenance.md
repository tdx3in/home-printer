# Maintain a Home Printer

Routine maintenance helps prevent print quality issues, paper jams, and failed
print jobs.

## Monthly Checklist

- [ ] Remove loose paper scraps from the input and output trays.
- [ ] Check ink or toner levels.
- [ ] Print a test page.
- [ ] Update printer firmware if the manufacturer recommends it.
- [ ] Clean the scanner glass with a soft cloth.

## Clean Print Heads

1. Open the printer maintenance menu.
2. Choose **Clean print head** or **Nozzle check**.
3. Follow the prompts.
4. Print a test page.
5. Repeat once if lines or gaps remain.

## Align the Printer

1. Open the printer maintenance menu.
2. Choose **Align printer** or **Print alignment page**.
3. Print the alignment page.
4. Scan the page if the printer asks for it.
5. Confirm that printed text and images look straight.

## Replace Supplies

Replace ink, toner, or maintenance cartridges when the printer reports low or
empty supplies. Use supplies that match the printer model.
