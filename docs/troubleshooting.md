# Troubleshoot a Home Printer

Use this guide to diagnose common printer problems.

## Printer Shows Offline

| Check | What to do |
| --- | --- |
| Power | Confirm that the printer is turned on. |
| Network | Confirm that the printer is connected to the same Wi-Fi network as the device. |
| Queue | Resume or clear the print queue. |
| Driver | Remove and re-add the printer if it does not respond. |

### Resolution Steps

1. Restart the printer.
2. Restart the computer or mobile device.
3. Confirm the Wi-Fi network name.
4. Print a network status page from the printer.
5. Remove and add the printer again if the device still shows offline.


## Poor Print Quality

| Symptom | Likely cause | Fix |
| --- | --- | --- |
| Faded text | Low ink or toner | Replace supply |
| Streaks | Dirty print head | Run print head cleaning |
| Misaligned lines | Alignment issue | Run printer alignment |
| Smudges | Wrong paper or wet ink | Use supported paper and allow drying time |

## Paper Jam

1. Turn off the printer.
2. Open the access panels.
3. Remove jammed paper slowly with both hands.
4. Check for torn paper pieces.
5. Reload the tray with flat, undamaged paper.
6. Turn on the printer and retry the job.

:::warning
Do not pull paper forcefully against the feed direction.
:::

## Escalate When

- The printer shows a hardware error code.
- The same paper jam returns after clearing the path.
- The printer cannot detect installed ink or toner.
- The power light does not turn on.
