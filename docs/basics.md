# Home Printer Basics

A home printer converts digital documents and images into printed pages. Most
modern printers also include scanning, copying, mobile printing, or cloud
printing features.

## Common Printer Types

| Type | Best for | Notes |
| --- | --- | --- |
| Inkjet | Photos, schoolwork, light home use | Uses ink cartridges or ink tanks. |
| Laser | Text-heavy documents | Uses toner and is usually faster for large jobs. |
| All-in-one | Printing, scanning, and copying | Combines multiple home-office tasks. |

## Key Terms

- **Printer driver**: Software that lets the computer communicate with the printer.
- **Print queue**: A list of documents waiting to print.
- **Duplex printing**: Printing on both sides of a sheet.
- **SSID**: The name of a Wi-Fi network.
- **WPS**: A router feature that can connect devices without typing the Wi-Fi password.

## Connection Options

- USB cable
- Wi-Fi network
- Ethernet cable
- Mobile printing app
- Cloud printing service

:::note
Wi-Fi setup is convenient, but USB is often the simplest option for a single computer.
:::
